## 简介

DeepWiki系统中的可视化图表功能是一个强大的代码分析和展示工具，它能够将复杂的代码结构和AI生成内容转化为直观的Mermaid格式交互式图表。该系统通过智能的提示词模板引导大型语言模型（LLM）输出符合Mermaid语法的图表代码，然后利用mermaid.js库在浏览器中实时渲染为高质量的SVG图形。

这一功能的核心价值在于：
- **自动化图表生成**：从代码分析结果自动生成架构图、流程图等可视化内容
- **实时渲染体验**：在浏览器中即时渲染复杂的图表，支持交互操作
- **多类型图表支持**：涵盖流程图、序列图、类图、实体关系图等多种图表类型
- **智能错误处理**：当AI输出无效语法时提供优雅的降级显示方案
- **主题适配能力**：支持日式美学风格和深色模式切换

## 图表类型与语法规范

### 支持的图表类型

系统支持多种类型的Mermaid图表，每种都有其特定的应用场景和语法规则：

#### 流程图（Flowchart）

流程图是最常用的图表类型，用于展示系统的处理流程和决策路径：

```mermaid
flowchart TD
A[开始] --> B{条件判断}
B --> |是| C[执行操作A]
B --> |否| D[执行操作B]
C --> E[结束]
D --> E
style A fill:#e1f5fe
style B fill:#fff3e0
style C fill:#e8f5e8
style D fill:#fce4ec
style E fill:#f3e5f5
```

#### 序列图（Sequence Diagram）

序列图用于展示对象之间的交互顺序和消息传递：

```mermaid
sequenceDiagram
participant User as 用户
participant App as 应用程序
participant API as 后端API
participant DB as 数据库
User->>App : 发起请求
App->>API : 验证身份
API->>DB : 查询数据
DB-->>API : 返回结果
API-->>App : 处理响应
App-->>User : 显示结果
Note over User,DB : 异步事件处理
App-)API : 异步通知
```

#### 类图（Class Diagram）

类图用于展示面向对象系统中的类、接口及其关系：

### 语法规范详解

#### 基本语法规则

| 规则类别 | 具体要求 | 示例 |
|---------|----------|------|
| 图表声明 | 必须以图表类型开头 | `flowchart TD` |
| 节点定义 | 使用方括号包围节点名称 | `A[开始]` |
| 连接线 | 使用`-->`或`->>`表示连接 | `A --> B` |
| 注释 | 使用`%%`添加注释 | `%% 这是注释` |
| 样式定义 | 使用`style`关键字设置样式 | `style A fill:#color` |

#### 箭头类型规范

系统支持多种箭头类型，每种都有特定的语义：

```mermaid

A )--|回调| E[响应]
style A fill:#ff9999
```

#### 布局方向要求

系统强制要求所有图表使用垂直布局（顶部到底部），这是为了保持一致的视觉体验：

```mermaid

A[输入] --> B[处理]
B --> C[输出]
%% 错误：水平布局被禁止
%% A --> B
%% B --> C
```
